Демо сайт для сервиса levam.ru

----------------------------------------
Для работы данного сайта вам необходимо:
----------------------------------------
1) PHP версии 5.5 или выше.
2) Модуль php-cURL - свободная кроссплатформенная служебная программа командной строки,
   позволяющая взаимодействовать с множеством различных серверов по множеству различных протоколов с синтаксисом URL.
3) Apache - HTTP-сервер.
4) mod_rewrite - модуль веб сервера Apache.
5) В файле config.php добавить свой ключ api
